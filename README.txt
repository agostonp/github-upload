Files in the package:
"Ago Udagram Project High Availability Architecture.jpeg" - Architecture Diagram
udagram-network.yml - CloudFormation template containing the Network elements
udagram-net-parameters.json - parameters to udagram-network.yml
udagram-servers.yml - CloudFormation template containing the servers and services
udagram-server-parameters.json - parameters to udagram-servers.yml
udagram.zip - the sample "Udagram Application" stored in S3 bucket and downloaded and extracted at startup on every AppServer
